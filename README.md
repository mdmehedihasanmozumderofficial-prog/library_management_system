#Library Management System

A simple **Library Management System** built with **Python** using basic programming concepts like **Functions, Dictionary, Conditional Statements, Loops, and User Input**.

This project allows users to manage books in different categories, borrow and return books, search for books, and display all available books.



##Features

- Add a new book
- Borrow a book
- Return a book
- Search for a book
- Show all books
- Show books by category
- Exit the program


##Technologies Used

- Python 3
- Dictionary
- Functions
- If-Else Statements
- Loops
- User Input


##How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Library-Management-System.git
```

2. Open the project folder

```bash
cd Library-Management-System
```

3. Run the program

```bash
python library_management.py
```
